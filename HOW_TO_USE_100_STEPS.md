## 📋 사용 방법

### 1단계: 현재 상태 확인
```bash
cd /home/user/webapp
bash scripts/next-step.sh
```

### 2단계: "ㄱ" 입력

채팅창에 **"ㄱ"** 만 입력하세요.

### 3단계: AI가 자동으로

1. **분석**: 현재 코드베이스 진단
2. **계획**: 해당 단계 작업 계획 수립
3. **구현**: 실제 코드 작성 및 테스트
4. **검증**: 빌드 및 기능 확인
5. **커밋**: Git commit 및 진행 상황 업데이트

### 4단계: 반복

다시 "ㄱ" 입력 → 다음 단계 진행

