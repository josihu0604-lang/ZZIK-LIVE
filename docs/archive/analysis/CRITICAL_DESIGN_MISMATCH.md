| 항목 | 원래 설계 (README.md) | 현재 구현 (/feed) |
|------|----------------------|-------------------|
| **핵심 가치** | GPS×QR×영수증 삼중 검증 | 인플루언서 콘텐츠 피드 |
| **타겟 시장** | 로컬 비즈니스 (B2B) | 일반 사용자 (B2C) |
| **수익 모델** | 월 구독 + 성과 수수료 | 불명확 (광고?) |
| **차별화** | 위치 검증 기술 | 없음 (일반 SNS) |

### 2. 네비게이션 구조 붕괴

```
✅ 설계된 구조:
app/(tabs)/
  ├── explore/  → 지도 기반 탐색
  ├── offers/   → 오퍼/할인
  ├── scan/     → QR 스캐너
  └── wallet/   → 포인트/보상

❌ 현재 구조:
app/
  ├── feed/     → 인플루언서 피드 (새로 추가)
  └── (tabs)/   → 기존 탭 방치/미사용
```

### 3. 사용자 플로우 변경

```
✅ 원래: 로그인 → 둘러보기 → /pass → 지도 탐색
❌ 현재: 로그인 → 둘러보기 → /feed → 인플루언서 피드
```
